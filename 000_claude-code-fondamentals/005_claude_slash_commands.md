# Claude code Slash commands

They are many. We will see some important ones

## /help
This shows help and available commands

## /compact
This command free up context by summarizong the conversation so far

## /clear
- Starts a new session with empty context;
- Previous session stays on disk (resume with /resume)

## /rewind
Restore the code and/or conversations to a previous point in the session

## /context
Visualize current context usage as a colored grid

## /config
This open settigns (Status, configs, Usage and Stats)

## /init
Initialize a new CLAUDE.md file with codebase documentation

## /doctor
Diagnose and verify your Claude Code installation and settings

## /memory
Create or edit CLAUDE.md file: User memory for all projets(~/.claude/CLAUDE.md) or Project memory(./CLAUDE.md) or auto-memory folder. 

## /exit
To quit
